# Upgrade Guide — vNEXT

<!-- bump: patch -->

## What Changed

- Fixed MessageSentinel LLM classifier misclassifying conversational "hold on" messages as pause commands. Messages like "OK hold on I don't think that's how it works" were being treated as pause directives when they're actually conversational disagreements. The LLM prompt now includes disambiguation guidance: if substantive content follows a hold-like phrase, it's classified as normal conversation.

## What to Tell Your User

- **Smarter message classification**: "Your agent is now better at understanding when you say 'hold on' as part of a thought versus when you're actually telling it to pause. Saying 'hold on, I disagree' will no longer accidentally pause your session."
