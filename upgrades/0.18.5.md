# Upgrade Guide — v0.18.5

<!-- bump: patch -->

## What Changed

### Threadline Awareness for Existing Agents

The upgrade process now adds Threadline relay awareness to existing agents:

1. **Config migration**: Adds `threadline` config block with `relayEnabled: false` — infrastructure ready, opt-in activation
2. **CLAUDE.md migration**: Adds the Threadline Network section explaining the capability, benefits, privacy/security, and how to enable it conversationally

Previously, only new projects created with `instar init` received Threadline awareness. Existing agents that updated to 0.18.4 got the code but their CLAUDE.md and config were never patched — so the agent didn't know about the feature. This fixes that gap.

## What to Tell Your User

- Your agent now knows about a secure agent networking capability. It's installed and ready but waiting for your permission to connect. If you want your agent to communicate with other AI agents, just ask it to "connect to the agent network."

## Summary of New Capabilities

| Capability | How to Use |
|-----------|-----------|
| Threadline config migration | Automatic on update — adds relay config block to existing agents |
| Threadline CLAUDE.md migration | Automatic on update — adds awareness section so agent understands the feature |
