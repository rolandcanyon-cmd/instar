# Upgrade Guide — v0.25.5

<!-- bump: patch -->

## What Changed

- **Bootstrap file threshold**: Session bootstrap messages (CONTINUATION context with thread history) that exceed 500 characters are now written to a temp file and injected as a file reference, instead of being passed directly to `tmux send-keys -l`. This fixes a bug where large bootstrap messages (~19KB) would fail silently during tmux injection, causing sessions to spawn idle and enter a zombie-kill-respawn loop.

- **Research navigation context**: Added `researchNavigationTemplate()` to ContextHierarchy, providing canonical source lookup guidance for research-oriented context segments.

## What to Tell Your User

- **Session stability fix**: "Sessions that were getting stuck in a restart loop when resuming conversations with long history should now work reliably. The context gets saved to a file and read in, instead of being crammed through a terminal pipe that was choking on large messages."
