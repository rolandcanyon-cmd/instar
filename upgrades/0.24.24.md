# v0.24.24 — Slack Standby Updates

## What Changed

**PresenceProxy wired to Slack** — Slack sessions now receive the same tiered standby updates (🔭) that Telegram gets. When a user sends a message in Slack and the agent takes time to respond, they see real-time status updates about what the agent is doing. Uses a synthetic ID bridge to map Slack channel IDs into the existing PresenceProxy without refactoring its internals.

## What to Tell Your User

When you send a message in Slack and the agent is working on it, you will now see status updates showing what the agent is doing. These appear as messages prefixed with a telescope icon. Previously this only worked in Telegram.

## Summary of New Capabilities

| Capability | How to Use |
|-----------|-----------|
| Slack standby updates | Automatic — fires when agent takes more than 20 seconds to respond in a Slack channel |
